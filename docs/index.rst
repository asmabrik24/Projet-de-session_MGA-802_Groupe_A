Fusion GPS/IMU pour la navigation véhiculaire à l’aide de Python

Projet réalisé dans le cadre du cours MGA802.

Équipe
-------

* Asma Brik
* Nada Chaouachi
* Brahim Rezgui

Description
-----------

Ce projet vise à développer une bibliothèque Python permettant :

* l'estimation de trajectoires GPS ;
* l'estimation de trajectoires IMU ;
* la fusion GPS/IMU ;
* la simulation de pannes GPS ;
* le calcul des erreurs de position ;
* le calcul du RMSE ;
* la génération de graphiques de validation.

Modules principaux
------------------

* chargeur_donnees.py
* pipeline.py
* pre_traitement.py
* navigation.py
* gps_outage.py
* evaluator.py
* user_interface.py
* visualization.py
