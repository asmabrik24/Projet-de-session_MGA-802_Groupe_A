Fusion GPS/IMU pour la navigation véhiculaire à l’aide de Python
================================================================

Projet réalisé dans le cadre du cours MGA802.

Équipe
-------

* Asma Brik
* Nada Chaouachi
* Brahim Rezgui

Description
-----------

Ce projet développe une bibliothèque Python permettant :

* l'estimation de trajectoires GPS ;
* l'estimation de trajectoires IMU ;
* la fusion GPS/IMU ;
* la simulation de pannes GPS ;
* le calcul des erreurs de position ;
* le calcul du RMSE ;
* la génération de graphiques de validation.
